## Instalation

Configure .env with .env.example.
Generate key:

```
php artisan key:generate
```

Install dependencies:

```
composer install
```

```
npm install
```

Create tables in database:

```
php artisan migrate
```

Use this 2 command together for strat it up:

```
php artisan serve
```

```
npm run dev
```

## Technology

Laravel 12, php 8, Bootstrap 5, Inertia.js, Vue.js 3
